# infra/

This directory contains infrastructure as code and deployment configurations.

## Contents

This directory typically includes:

- Cloud infrastructure definitions (Terraform, CloudFormation, etc.)
- Docker and containerization files
- Kubernetes manifests
- CI/CD pipeline configurations
- Environment-specific configurations
- Deployment scripts

## Organization

Organize by infrastructure type or environment:

```
infra/
  terraform/     - Terraform configurations
  docker/        - Dockerfiles and docker-compose files
  k8s/          - Kubernetes manifests
  environments/ - Environment-specific configs
```
