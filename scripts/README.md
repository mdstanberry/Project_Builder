# scripts/

This directory contains utility and automation scripts.

## Purpose

Scripts in this directory are used for:

- Build and deployment automation
- Data migration and setup
- Development utilities
- Testing helpers
- Code generation

## Best Practices

- Include a shebang line for shell scripts (`#!/bin/bash` or `#!/usr/bin/env python3`)
- Add comments explaining what each script does
- Document required parameters and usage
- Make scripts executable when appropriate
