# src/

This directory contains source code for single-service backend projects.

## Structure

For single-service backends, organize code as follows:

- `src/config/` - Configuration files and environment setup
- `src/routes/` - API routes and endpoints
- `src/services/` - Business logic and service layer
- `src/models/` - Data models and schemas
- `src/lib/` - Shared utilities and helper functions

## Usage

Use this directory when building a single-service application (API, microservice, etc.).
